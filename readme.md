#JitterBot
__________________

Discord Bot `version 0.1`


Discord Channel: https://discord.gg/Ndrv5ftWzW 

##Install

1.Clone repository
```
git clone git@github.com:gjhonic/JitterBot.git
```

2. move to JitterBot dir
```
cd JitterBot
```

3. update composer
```
composer update
```

4. create .env
```
.sample.env > .env
```

5. set param in .env


6. start bot
```
php index.php
```
